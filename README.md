# Snake

Ce projet a pour but de créer un jeu de snake en C# avec Unity.

## Features

-  3 Scenes
  - Menu
  - Game
- 3 Difficultés
  - Easy
  - Hard
  - Impossible
- Gestions des scores
- Gestion des sons
- Écran de fin de partie
- Gestion des collisions
- Ramassage des pommes
